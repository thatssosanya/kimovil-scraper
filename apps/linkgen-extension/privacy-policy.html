Privacy Policy for LinkGen Chrome Extension

Last updated: January 3, 2025

1. Introduction

This privacy policy describes how the LinkGen Chrome Extension ("we", "our", or "the extension") collects, uses, and protects your information when you use our browser extension for generating affiliate links for Yandex.Market, AliExpress, and Price.ru.

2. Information We Collect

2.1 Data Stored Locally

The extension stores the following data locally on your device using Chrome's storage API:

  • Generated Links: Cached affiliate links you've generated (stored for 24 hours)
  • Author Selection: Your selected author/VID preference for Yandex.Market links
  • Tab Information: Current tab URL and title for link generation purposes

2.2 Data Accessed

The extension accesses:

  • Active Tab URL: To detect supported e-commerce sites and generate appropriate affiliate links
  • Tab Title: To display product information in the extension interface

3. How We Use Your Information

We use the collected information solely for:

  • Generating affiliate links for supported e-commerce platforms
  • Caching generated links to improve performance
  • Remembering your author preferences
  • Checking AliExpress commission rates through our API

4. Data Sharing and Third Parties

4.1 External Services

The extension communicates with the following external services:

  • kik.cat: URL shortening service for generating short affiliate links
  • c.click-or-die.ru: API service for checking AliExpress commission rates and generating deeplinks

4.2 Data Transmitted

When using the extension, the following data may be sent to external services:

  • Product URLs from supported e-commerce sites
  • Generated affiliate links for shortening
  • No personal information or browsing history outside of the current product page

5. Data Storage and Retention

  • Local Storage: All user data is stored locally on your device
  • Cache Expiration: Generated links are automatically deleted after 24 hours
  • No Cloud Storage: We do not store any user data on our servers
  • Uninstall: All data is removed when you uninstall the extension

6. Data Security

We implement appropriate security measures:

  • All data is stored locally using Chrome's secure storage API
  • External API communications use HTTPS encryption
  • No sensitive personal information is collected or transmitted

7. Your Rights and Choices

You have the right to:

  • Access: View all locally stored data through the extension interface
  • Delete: Clear cached data through the extension or by uninstalling
  • Control: Choose which author/VID to use for link generation
  • Opt-out: Stop using the extension at any time

8. Children's Privacy

This extension is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

9. Changes to This Privacy Policy

We may update this privacy policy from time to time. The "Last updated" date at the top of this policy indicates when it was last revised. We encourage you to review this policy periodically.

10. Data Processing Lawful Basis

Our processing of your data is based on:

  • Legitimate Interest: Providing affiliate link generation functionality
  • Consent: Your voluntary use of the extension
  • Performance of Service: Delivering the core functionality you requested

11. International Data Transfers

While data is primarily stored locally on your device, external API services may be located in different countries. We ensure that any international data transfers comply with applicable privacy laws.
